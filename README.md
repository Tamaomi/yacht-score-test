# ヨット点数計算表

## アプリの概要
Androidのホーム画面に追加して使える、ダイスゲーム「ヨット」用の点数計算PWAです。2人用・3人用・4人用の画面を切り替えて使えます。

## ファイル構成
- index.html
- style.css
- script.js
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

## Android Chromeでの開き方
1. GitHub Pagesなどにファイル一式をアップロードします。
2. Android Chromeで公開URLを開きます。
3. 画面が表示されたら使用できます。

## ホーム画面への追加方法
1. Android Chrome右上の「︙」を押します。
2. 「ホーム画面に追加」を押します。
3. 名前を確認して「追加」を押します。
4. ホーム画面のアイコンから起動できます。

## オフライン利用方法
一度オンラインで開くと、必要ファイルがキャッシュされます。以後は通信なしでも起動できます。

## 更新方法
ファイルを差し替えたあと、Chromeでページを再読み込みしてください。反映されない場合はキャッシュ削除を行ってください。

## キャッシュ削除方法
Android Chromeの設定から、サイト設定または閲覧履歴データの削除を開き、該当サイトのデータを削除してください。

## よくあるトラブル
- ホーム画面追加が出ない：HTTPSで公開されているか確認してください。
- 更新が反映されない：ChromeのキャッシュまたはPWAのキャッシュを削除してください。
- オフラインで開けない：一度オンライン状態でページを最後まで読み込んでください。

## v3 修正版の変更内容

- ダイスのアイコン表記と、添付画像に近い表形式のデザインを維持しました。

## 勝敗判定
現在選択している人数の全プレイヤーについて、12役すべてのスコアが入力されると、画面下部に「勝敗判定」ボタンが表示されます。ボタンを押すと、指定画像と「ホンダの勝ち！！」を表示します。
