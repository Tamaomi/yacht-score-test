# ヨット点数計算表

## アプリの概要
ダイスゲーム「ヨット」の点数をスマホで入力・計算する個人利用向けPWAです。Android Chromeでホーム画面に追加できます。

## ファイル構成

```text
index.html
style.css
script.js
manifest.json
service-worker.js
README.md
icons/
  icon-192.png
  icon-512.png
```

## Android Chromeでの開き方
1. GitHub Pagesなどにこのフォルダの中身をアップロードします。
2. Android Chromeで公開URLを開きます。
3. 画面が表示されたら通常のWebページとして使用できます。

## ホーム画面への追加方法
1. Android ChromeでアプリのURLを開きます。
2. 右上の「︙」を押します。
3. 「ホーム画面に追加」または「アプリをインストール」を押します。
4. 追加後、ホーム画面のアイコンから起動できます。

## オフライン利用方法
初回だけオンラインで開いてください。以後はService Workerにより、通信なしでも起動できます。

## 更新方法
1. 修正したファイルをGitHubなどへ再アップロードします。
2. Android Chromeでページを再読み込みします。
3. 古い表示が残る場合はキャッシュを削除します。

## キャッシュ削除方法
Android Chromeで対象サイトを開き、サイト設定から保存データを削除してください。GitHub Pagesの場合はURLを開き直して再読み込みしてください。

## よくあるトラブル

### ホーム画面に追加が出ない
manifest.json と service-worker.js が同じ階層にあるか確認してください。HTTPSで公開されている必要があります。

### オフラインで開けない
初回起動時にオンラインで一度開いていない可能性があります。オンラインで一度表示してから再確認してください。

### 更新したのに古い画面が出る
Service Workerのキャッシュが残っています。Chromeのサイトデータ削除後に再読み込みしてください。

### 点数が大きく入力できない
各役の想定最大点を超える入力は自動で補正されます。
